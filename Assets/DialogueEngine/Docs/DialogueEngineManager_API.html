DialogueEngineManager API 문서

기준 파일: Assets/DialogueEngine/Scripts/DialogueEngineManager/DialogueEngineManager.cs

DialogueEngineManager는 대화 시스템 런타임 진입점입니다. 초기화, 대화 시작, 라이프사이클 등록, 이벤트 핸들러 등록, 파라미터 값을 외부에서 제어할 수 있습니다.

Static API Runtime Control Parameter Binding

빠른 시작

using DialogueEngine;

public class DialogueBootstrap : MonoBehaviour
{
    private void Start()
    {
        DialogueEngineManager.Init(success =>
        {
            if (!success) return;
            DialogueEngineManager.StartDialogue("INTRO_001");
        });
    }
}

권장 순서: Init 완료 후 StartDialogue 호출

Method Details

Init(Action<bool> onComplete)

엔진 리소스를 비동기로 초기화하고 결과를 콜백으로 반환합니다.

ParameterTypeDescription
onCompleteAction<bool>초기화 성공 시 true, 실패 시 false
 
DialogueEngineManager.Init(success =>
{
    if (!success) Debug.LogError("Init failed");
});

StartDialogue(string dialogueId)

dialogueId를 기준으로 대화를 시작합니다.

ParameterTypeDescription

이벤트 이름에 사용자 코드를 연결합니다. null을 전달하면 등록을 해제합니다.

주의사항

  • Init 전에 StartDialogue를 호출하면 동작하지 않을 수 있습니다.
  • SetTimeParameter 값은 hh:mm:ss 형식을 권장합니다.
  • 파라미터 이름/타입은 Settings에 정의된 항목과 일치해야 합니다.
